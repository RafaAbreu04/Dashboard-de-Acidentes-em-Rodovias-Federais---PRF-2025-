# Dashboard de Acidentes em Rodovias Federais - PRF (2025)

> Meu primeiro projeto utilizando Power BI, desenvolvido com o objetivo de praticar ETL, modelagem de dados e criação de dashboards a partir de dados públicos.

![Dashboard](Dashboard - AcidentesPRF2025)

---

## 📖 Sobre o projeto

Este projeto foi desenvolvido como parte dos meus estudos em Análise de Dados.

Utilizei a base pública de Acidentes em Rodovias Federais disponibilizada pela Polícia Rodoviária Federal (PRF) no Portal de Dados Abertos do Governo Federal para praticar todo o processo de desenvolvimento de um dashboard, desde a importação dos dados até a construção das visualizações.

O projeto surgiu com o objetivo de colocar em prática conhecimentos que venho estudando sobre Power BI, Power Query e ETL, além de desenvolver uma forma clara e intuitiva de apresentar informações.

---

## 🎯 Objetivos

- Praticar ETL utilizando Power Query;
- Trabalhar com dados públicos;
- Desenvolver um dashboard interativo no Power BI;
- Criar indicadores (KPIs);
- Melhorar minhas habilidades em visualização de dados;
- Aplicar conceitos de Análise de Dados em um projeto real.

---

## 📂 Fonte dos dados

Os dados utilizados neste projeto são públicos e podem ser encontrados no Portal de Dados Abertos do Governo Federal.

Base utilizada:

- Acidentes em Rodovias Federais – PRF (2025) (agrupados por ocorrência)

---

## 🛠️ Ferramentas utilizadas

- Power BI Desktop
- Power Query
- DAX
- GitHub

---

## 🔄 Processo de desenvolvimento

### 1. Importação

- Importação do arquivo CSV para o Power BI.

### 2. ETL (Power Query)

Durante a etapa de tratamento foram realizadas transformações como:

- Limpeza dos dados;
- Alteração de tipos das colunas;
- Tratamento de valores nulos;
- Organização das informações;
- Criação de colunas auxiliares.

### 3. Modelagem

Após o tratamento da base, os dados foram preparados para facilitar a construção dos indicadores e visualizações.

### 4. Dashboard

Foram desenvolvidos indicadores e gráficos com foco em facilitar a análise dos acidentes registrados nas rodovias federais.

---

## 📊 Indicadores (KPIs)

O dashboard apresenta:

- Total de acidentes;
- Total de feridos;
- Total de mortos;
- Acidentes graves;
- Mortes a cada 100 acidentes.

---

## 📈 Visualizações

O painel contém análises sobre:

- Acidentes por mês;
- Acidentes por horário;
- Top 10 estados com mais acidentes;
- Top 10 causas de acidentes.

## 🚧 Desafios encontrados

Durante o desenvolvimento encontrei alguns desafios, entre eles:

- Definir quais indicadores eram mais relevantes;
- Organizar os elementos do dashboard de forma intuitiva;
- Tratar inconsistências na base de dados;
- Transformar dados brutos em informações úteis para análise.
