# Dashboard-de-Acidentes-em-Rodovias-Federais---PRF-2025-

## Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar técnicas de ETL, modelagem de dados e construção de dashboards utilizando Power BI.

Os dados utilizados são públicos e foram obtidos no Portal de Dados Abertos do Governo Federal, contendo registros de acidentes em rodovias federais da Polícia Rodoviária Federal (PRF) referentes ao ano de 2025.
